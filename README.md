* `pkg update`
* `pkg upgrade`
* `pkg install git `
* `git clone https://github.com/mishakorzik/Ultra-DDos`

#### Finished you have successfully downloaded Ultra-DDos now to launch!

* `cd Ultra-DDos`
* `python2 ddos.py`
